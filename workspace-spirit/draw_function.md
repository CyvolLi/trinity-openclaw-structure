# draw_function.md — 画图功能定义

## 1. 功能描述
根据用户提供的文字描述（与参考上下文），调用 AI 图像生成能力绘制图片，返回图片供用户查看/保存。

## 2. 工作文件夹
Son 工作区内路径: draw/

## 3. 触发指令
| 指令 | 参数 | 说明 |
|------|------|------|
| `/画图 <描述>` | 描述：对画面的文字描述（支持中文、英文） | 根据文字描述生成一幅图片 |
| `/画图 <描述> --风格 <风格>` | 描述 + 风格选项（如：二次元、写实、水彩、油画、像素风） | 按指定风格生成图片 |
| `/画图 <描述> --尺寸 <宽x高>` | 描述 + 尺寸（如 1024x1024、1920x1080） | 按指定分辨率生成图片 |
| `/画图 <描述> --参考 #图片ID` | 描述 + 参考已有生成的图片 ID 作为构图参考 | 参考已有图片风格/构图生成新图 |

## 4. 执行接口

### 4.1 生成图片
Son 执行步骤:
1. 解析用户消息，提取描述文字、风格参数、尺寸参数、参考图 ID（如有）
2. 若指定了参考图 ID，在 draw_history.db 中查找对应图片文件路径
3. 调用图像生成模型（默认文生图；有参考图时图生图），携带参数：
   - prompt: 用户文字描述
   - style: 风格（如有）
   - size: 尺寸（如有，默认为 1024x1024）
   - reference_image: 参考图路径（如有）
4. 接收生成的图片，保存到 draw/ 目录，命名格式：draw_{id}.png
5. 在 draw_history.db 中插入生成记录
6. 返回图片给 Father，附带生成描述信息

### 4.2 查看历史
Son 执行步骤:
1. 用户发送 `/画图历史` 或 `/看图记录`
2. 在 draw_history.db 中按时间倒序查询最近 10 条记录
3. 返回记录列表（ID + 描述前20字 + 风格 + 尺寸 + 生成时间）

### 4.3 重新生成
Son 执行步骤:
1. 用户发送 `/重画 #图片ID <修改描述>`
2. 在 draw_history.db 中根据 ID 查找原记录
3. 使用原 prompt + 新修改描述 合并作为新 prompt 重新生成
4. 保存新图，记录新记录（关联原 ID 到 parent_id 字段）
5. 返回新图片

## 5. 数据库字段说明

### 表名
draw_history

### 字段说明
| 字段 | 类型 | 必填 | 说明 |
|------|------|:--:|------|
| id | TEXT | 是 | 唯一图片 ID，见下方生成规则 |
| prompt | TEXT | 是 | 用户输入的原始描述文字 |
| style | TEXT | 否 | 风格选项（如二次元、写实等），留空表示默认 |
| size | TEXT | 否 | 图片尺寸（如 1024x1024），留空表示默认 |
| reference_id | TEXT | 否 | 参考图片的 ID（如有） |
| parent_id | TEXT | 否 | 重新生成时关联的原始图片 ID |
| image_path | TEXT | 是 | 保存的图片文件路径（相对 Son 工作区） |
| model_used | TEXT | 否 | 使用的图像生成模型名称 |
| created_at | TEXT | 是 | 生成时间（ISO 8601 格式，如 2026-05-31T19:50:00+08:00） |

### ID 生成规则
格式: `DRAW-YYYYMMDD-NNNNN`
- `DRAW` 固定前缀
- `YYYYMMDD` 为生成日期（北京时间）
- `NNNNN` 为当天序号，从 00001 开始，自增

## 6. Son 返回格式规范

### 生成图片成功
```
🎨 画好啦！ID: DRAW-20260531-00001
📝 描述: {prompt}
🎭 风格: {style | 默认}
📐 尺寸: {size | 1024x1024}
{附图片}
```

### 生成图片失败
```
😵 画图失败：{失败原因}
试试换个描述或稍后再试~
```

### 查看历史（无记录时）
```
📭 还没有生成过图片哦，试试 /画图 <描述> 来画第一张吧！
```

### 查看历史（有记录时）
```
🖼️ 画图历史（最近10条）:

DRAW-20260531-00001 — "夕阳下的海边小屋" | 写实 | 1024x1024 | 2026-05-31 19:50
DRAW-20260531-00002 — "一只戴帽子的猫" | 二次元 | 512x512 | 2026-05-31 19:55
...
```

### 重新生成成功
```
🔄 重新画好了！新 ID: DRAW-20260531-00003
📝 原描述: {原始prompt}
🖊️ 修改: {修改描述}
{附图片}
```
