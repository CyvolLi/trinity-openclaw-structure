## 1. 功能描述

恶臭数字：将用户输入的目标数字（如出生日期、纪念日等），使用 114514 系列数字组合（1、5、4、11、14、15、51、54、114、514、1145、1451、14514、114514、1145140 等）通过加、减、乘、除、乘方、阶乘、拼接等数学运算拆解成恶臭风格的数学表达式。灵感来源于 lab.magiconch.com 上的数字恶臭化工具。

## 4. 执行接口

### 4.1 /恶臭 或 /臭数字

Son 执行步骤：

1. **参数提取与校验**
   - 从用户输入中提取目标数字 N（正整数）
   - 如果 N ≤ 0 或非数字，返回错误提示
   - 如果 N > 10^9，提示数字过大，降低范围

2. **可用数字库定义**
   ```
   基础数集: [1, 5, 4, 11, 14, 15, 51, 54]
   扩展数集: [114, 514, 1145, 1451, 14514, 114514, 1145140]
   特殊数: [1919810, 1145141919810]（更大数值用）
   ```
   - 根据 N 的大小，选择合适范围的数集（N 较小时只用基础数集）

3. **生成恶臭表达式（算法）**
   Son 使用以下算法生成至少 3 种不同的表达式：

   **算法 A — 加减拆解**
   - 将 N 拆分为多个基础数的和/差
   - 例：N = 2000 → 1145 + 514 + 114 + 114 + 114 - 1
   - 尽可能使用 114 和 514 的倍数

   **算法 B — 乘法组合**
   - 将 N 表示为两个或多个基础数的乘积，再加减修正
   - 例：N = 2026 → 5 × 514 - 114 × 4 + 5 + 5 - 4 + ...
   - 优先使用 114 × 514 = 58596 作为大数基准

   **算法 C — 平方/阶乘/幂组合**（N 较大时）
   - 使用 (a×b)²、a! 等运算扩大量级
   - 例：N = 200000 → (514 + 5)² × 4 + 1145 × 14 + ...

   **算法 D — 拼接拆解**（当 N 本身包含 114514 系列数字时）
   - 如果 N 的十进制表示中包含 114、514、1145 等序列，直接保留
   - 例：N = 114514 → "114514"（完美恶臭！）

4. **选择最佳表达式**
   - 按以下标准排序选择 3-5 个：
     - 优先使用 114 和 514 出现次数多的
     - 优先使用基础数集多、异类数少的
     - 优先运算简单的

5. **格式美化**
   - 每个表达式转为人类可读格式：用 ×、÷、＋、－ 符号
   - 添加emoji装饰：🤢 开头
   - 对最优表达式额外奖励 📿 标记

6. **存入数据库（可选）**
   - `INSERT INTO queries (input_number, expressions, requester_id, created_at) VALUES (?, ?, ?, ?)`
   - 缓存用户查询记录供后续参考

7. **返回结果给 Father**

### 边缘情况处理

| 情况 | 处理方式 |
|------|---------|
| 参数为空 | 返回："🤢 你要恶臭哪个数字？试试 /恶臭 114514" |
| 参数非数字 | 返回："🤢 请输入一个正整数，如 /恶臭 520" |
| 数字 ≤ 0 | 返回："🤢 0 和负数也能恶臭，但你先给个正整数吧" |
| 数字过大（>10^9） | 返回："🤢 数字太大了，暂时只支持 10 亿以内的数字" |
| 目标数字就是 114514 | 返回："🤢 114514 本身就是恶臭之巅，完美！" |

### 恶臭等级评分规则

Son 在返回每个结果时，必须自动计算 **恶臭等级**（1-10 级），评分规则如下：

| 因子 | 加分规则 | 最高分 |
|------|---------|:-----:|
| **114 出现次数** | 每次 +2 分 | +6 |
| **514 出现次数** | 每次 +3 分 | +9 |
| **114514 完整出现** | 每次 +5 分 | +10 |
| **运算复杂度** | 仅 +−×÷ → +1；含幂/阶乘 → +3 | +3 |
| **表达式长度** | 超过 5 个运算符 → +1 | +1 |
| **恶臭数占比** | 运算符间超过 70% 是恶臭数 → +2 | +2 |

**等级转换（将总分映射到 1-10 级）**：

| 总分范围 | 恶臭等级 | 显示 |
|:--------:|:-------:|------|
| 0-2 | 1-2/10 | 🫵 不够臭啊 |
| 3-5 | 3-4/10 | 💨 微微发臭 |
| 6-8 | 5-6/10 | 🤢 有点味道了 |
| 9-12 | 7-8/10 | 💩 恶臭来袭 |
| 13-16 | 9/10 | 🔥 非常恶臭 |
| 17+ | 10/10 | 🏆 究极恶臭！ |

**特殊加分**：
- 目标数字本身就是 114514 → 直接 10/10 🏆 究极恶臭！
- 表达式仅由 114 和 514 拼接而成 → 额外 +5
